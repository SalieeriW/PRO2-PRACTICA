\item\contentsline{section}{\mbox{\hyperlink{class_area__de___espera}{Area\+\_\+de\+\_\+\+Espera}} \\*Hace referencia a la area de espera de los procesos. Contiene procesos que se quieren tratar en el cluster classificados por prioridad. Puede usarse para crear o destruir una prioridad, poner un proceso en espera, dar de alta a procesos al cluster, consultar la existencia de prioridades y procesos, consultar si una prioridad tiene tareas pendientes y dispone de operaciones d\textquotesingle{}escriptura para una o todas las prioridades }{\pageref{class_area__de___espera}}{}
\item\contentsline{section}{\mbox{\hyperlink{class_cluster}{Cluster}} \\*Esta clase hace referencia a un \mbox{\hyperlink{class_cluster}{Cluster}}, un conjunto de procesadores (minimo 1) que realizan procesos y tareas. Permite anadir procesadores al \mbox{\hyperlink{class_cluster}{Cluster}}, compactar la memoria de un procesador o todos los procesadores del \mbox{\hyperlink{class_cluster}{Cluster}}, dar de alta a procesos (directamente a un procesador determinado o al procesador más optimo. También permite hacer consultas sobre l\textquotesingle{}estado de un procesador, si existe, tiene tareas en ejecución, si tiene procesadores auxiliares y la memoria disponible. No obstante, dispone de un contador de tiempo, se puede avanzar el tiempo para terminar los procesos en ejecución. Por último, dispone de operaciones de lectura y escriptura a nivell de cluster y a nivell de procesador }{\pageref{class_cluster}}{}
\item\contentsline{section}{\mbox{\hyperlink{class_prioridad}{Prioridad}} \\*Esta clase hace referencia a una prioridad, un contenedor de procesos con una etiqueta formado por un string. Dispone de un conjunto de operaciones que permite anadir y eliminar procesos de una prioridad. Permite consultar el proceso mas antiguo, el numero de procesos acceptados y rechazados, el numero de procesos pendientes, si existe un proceso concreto. Por ultimo dispone de una operacion de escriptura que imprime todos los procesos de una prioridad }{\pageref{class_prioridad}}{}
\item\contentsline{section}{\mbox{\hyperlink{class_procesador}{Procesador}} \\*Clase de procesador, hace referencia a un procesador que puede ejercutar procesos. Dispone de operaciones para anadir o eliminar un proceso al procesador, compactar la memoria de un procesador, avanzar el reloj de un procesador. Tambíen permite hacer consultas sobre si un procesador está ocupado, la memoria disponible y la existencia de un proceso. Por ultimo, tiene operaciones de lectura y escriptura para leer los datos de un procesador y escribir los procesos de estos }{\pageref{class_procesador}}{}
\item\contentsline{section}{\mbox{\hyperlink{class_proceso}{Proceso}} \\*Clase de proceso, hace referencia a tareas que tienen un identificador, memoria requierida y tiempo de ejecución. Dispone de operaciones para consultar cada datos de un proceso. Permite leer y escribir los datos de cada proceso }{\pageref{class_proceso}}{}
