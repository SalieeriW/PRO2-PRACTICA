\hypertarget{class_cluster}{}\doxysection{Referencia de la Clase Cluster}
\label{class_cluster}\index{Cluster@{Cluster}}


Esta clase hace referencia a un \mbox{\hyperlink{class_cluster}{Cluster}}, un conjunto de procesadores (minimo 1) que realizan procesos y tareas. Permite anadir procesadores al \mbox{\hyperlink{class_cluster}{Cluster}}, compactar la memoria de un procesador o todos los procesadores del \mbox{\hyperlink{class_cluster}{Cluster}}, dar de alta a procesos (directamente a un procesador determinado o al procesador más optimo. También permite hacer consultas sobre l\textquotesingle{}estado de un procesador, si existe, tiene tareas en ejecución, si tiene procesadores auxiliares y la memoria disponible. No obstante, dispone de un contador de tiempo, se puede avanzar el tiempo para terminar los procesos en ejecución. Por último, dispone de operaciones de lectura y escriptura a nivell de cluster y a nivell de procesador.
