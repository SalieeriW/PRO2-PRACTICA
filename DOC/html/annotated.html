 CArea_de_EsperaHace referencia a la area de espera de los procesos. Contiene procesos que se quieren tratar en el cluster classificados por prioridad. Puede usarse para crear o destruir una prioridad e insertar un nuevo proceso a la espera
 CClusterEsta clase hace referencia a un Cluster un conjunto de procesadores (minimo 1) que realizan procesos y tareas. Dispone de un contador de tiempo y permite operar con los procesadores: asignar tareas (procesos) a partir de la area de espera o inserta de forma directa a un procesador del Cluster o bien compactar la memoria de estos
 CPrioridadEsta clase hace referencia a una prioridad, un contenedor de procesos con una etiqueta formado por un string que los une
 CProcesadorClase de procesador, hace referencia al procesador. Permite crear un procesador acompanado con su identificador. Permite realizar operaciones con los procesadores
 CProcesoClase de proceso, hace referencia a tareas que tienen un identificador, memoria requierida y tiempo de ejecución
