class  Cluster
 Es la clase principal, hace referencia al Cluster, una estructura de datos que contiene un conjunto de procesadores (minimo 1) organizados en forma de arbol binario. Dispone de un contador de tiempo y permite operar con los procesadores: asignar tareas (procesos) a partir de la area de espera o inserta de forma directa a un procesador del Cluster o bien compactar la memoria de estos.
