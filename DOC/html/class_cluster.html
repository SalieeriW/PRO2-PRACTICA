Detailed Description

Esta clase hace referencia a un Cluster un conjunto de procesadores (minimo 1) que realizan procesos y tareas. Dispone de un contador de tiempo y permite operar con los procesadores: asignar tareas (procesos) a partir de la area de espera o inserta de forma directa a un procesador del Cluster o bien compactar la memoria de estos.
