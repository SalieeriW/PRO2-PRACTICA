\label{class_procesador}\index{Procesador@{Procesador}}


Clase de procesador, hace referencia a un procesador que puede ejercutar procesos. Dispone de operaciones para anadir o eliminar un proceso al procesador, compactar la memoria de un procesador, avanzar el reloj de un procesador. Tambíen permite hacer consultas sobre si un procesador está ocupado, la memoria disponible y la existencia de un proceso. Por ultimo, tiene operaciones de lectura y escriptura para leer los datos de un procesador y escribir los procesos de estos.
